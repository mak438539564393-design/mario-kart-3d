# 3D Mario Kart Game

WebGLで作成した3Dマリオカートゲームです。オフラインで完全に動作します。

## 機能

- 🏎️ 3D車の操作とレーシング
- 🏁 ラップカウント機能
- 🎮 キーボード操作対応
- 🌳 装飾された3Dトラック
- 💨 速度とラップ表示

## 操作方法

- **↑ / W** : 前進
- **↓ / S** : 後退
- **← / A** : 左旋回
- **→ / D** : 右旋回
- **SPACE** : ブレーキ

## セットアップ

1. ファイルを配置する
2. `index.html`をブラウザで開く
3. すぐにゲームを楽しめます！

## 必要なファイル

- `index.html` - ゲームのメインHTML
- `game.js` - ゲームロジック
- `lib/three.min.js` - Three.jsライブラリ（オフライン用）

## ブラウザ対応

- Chrome
- Firefox
- Safari
- Edge

## ゲームの特徴

### トラック
- 楕円形の曲がったトラック
- 装飾的な木々
- フィニッシュライン
- ラップ検出機能

### 車両
- リアルなホイール回転
- 加速・減速・ブレーキ機能
- 慣性物理演算
- カメラ追従システム

### ゲームプレイ
- スピードメーター
- ラップカウンター
- 画面外バウンダリ
- スムーズなカメラ移動
